# Meu Projeto React

Este é um projeto React com TypeScript configurado com Vite.

## Instalação

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Execute o projeto com `npm run dev`

## Estrutura de Pastas

- src/
  - components/
  - pages/
  - hooks/
  - utils/
  - types/

## Scripts

- `dev`: Inicia o servidor de desenvolvimento
- `build`: Gera a build de produção
- `preview`: Visualiza a build de produção localmente

## Estilos

Os estilos são modularizados e mantidos em arquivos CSS separados.
