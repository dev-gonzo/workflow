## Projeto React com TypeScript

Este projeto é um exemplo de aplicação React desenvolvida com TypeScript, seguindo padrões de mercado.

### Configuração Inicial

Para executar o projeto localmente, siga os passos abaixo:

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Execute o projeto com `npm run dev`

### Padrões de Mercado

- Utilização do Vite como bundler
- Configuração do ESLint e Prettier para garantir a qualidade do código
- Componentes funcionais com TypeScript
- Utilização de hooks modernos (useState, useEffect, etc.)
- Estrutura de pastas: src/components, src/pages, src/hooks, src/utils, src/types
