# Meu Projeto
Este é um projeto simples para demonstrar um "Hello, World!" em HTML.
## Como executar
Para visualizar o "Hello, World!" em HTML, siga as etapas abaixo:
1. Clone o repositório para sua máquina local.
2. Abra o arquivo `index.html` em um navegador web.
## Exemplo de HTML Hello, World!
```html
<!DOCTYPE html>
<html>
<head>
    <title>Hello World</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
```
