# Meu Projeto React com TypeScript

Este é um projeto React criado com TypeScript, seguindo padrões de mercado.

## Estrutura de Pastas

- `src/components`: Componentes React
- `src/pages`: Páginas da aplicação
- `src/hooks`: Hooks customizados
- `src/utils`: Funções utilitárias
- `src/types`: Tipos TypeScript

## Scripts Disponíveis

### `npm run dev`

Inicia o servidor de desenvolvimento.

### `npm run build`

Compila o projeto para produção.

### `npm run preview`

Inicia um servidor local para pré-visualização do build de produção.

## Configurações

- ESLint e Prettier configurados
- Utilização de CSS Modules ou Styled Components
