# Meu Projeto
Este é um projeto simples para demonstrar um "Hello World!" em HTML.
## Hello World em HTML
Para exibir um "Hello World!" em HTML, você pode usar o seguinte código:
```html
<!DOCTYPE html>
<html>
<head>
    <title>Hello World</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
```
Basta copiar e colar o código acima em um arquivo HTML e abrir em um navegador para ver a mensagem "Hello World!" sendo exibida.
