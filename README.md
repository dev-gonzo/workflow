# Meu Projeto React com TypeScript

Este é um projeto React com TypeScript seguindo padrões de mercado.

## Como executar

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Execute o projeto com `npm run dev`

## Estrutura de Pastas

- src/components
- src/pages
- src/hooks
- src/utils
- src/types
