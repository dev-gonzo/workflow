# Meu Projeto React

Projeto React com TypeScript e configuração inicial seguindo padrões de mercado.

## Como executar o projeto

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Execute o projeto com `npm run dev`

## Estrutura de pastas

- src/
  - components/
  - pages/
  - hooks/
  - utils/
